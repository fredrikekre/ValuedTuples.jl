    ValuedTuples.@VT — Macro.
    @VT args...

    Make a ValuedTuple. A valued tuple can be indexed only with Vals (create with @value). Valued tuples can be manipulated in a type-stable way because the names are directly encoded into the type. You can use repeated values. getindex will take the last match when trying to index at a repeated value; for all matches, use match_index instead. A vector of NamedTuples with consistent names will conveniently print as a markdown table.

    julia> using ValuedTuples

julia> b = 2;

julia> v = @VT a = 1 b a = 3
(@VT a = 1 b = 2 a = 3)

julia> v[@value b]
2

julia> v[@value d]
ERROR: BoundsError: attempt to access ()
[...]

julia> v[@value a]
3
